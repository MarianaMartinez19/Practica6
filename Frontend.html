¿Qué es Frontend?

El frontend es la parte visual de una aplicación o sitio web. Es todo aquello con lo que el usuario interactúa directamente.

Fundamentos del Frontend

  • HTML: estructura del sitio web
  • CSS: diseño y presentación
  • JavaScript: interactividad y dinamismo

El objetivo del frontend es ofrecer una experiencia clara, atractiva y fácil de usar para el usuario.
